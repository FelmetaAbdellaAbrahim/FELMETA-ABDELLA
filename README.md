# Felmeta Abdella (full name Felmeta Abdella Abrehim; Oromo: Falmataa Abdallaa; ግእዝ፡ ፈልመታ አብደላ; born December 12, 2004) is an Ethiopian political activist and a member of the Oromo Federalist Congress (OFC) political party. He is known for his association with the Qeerroo movement, an Oromo youth movement prominent in recent Ethiopian political changes.
