# FELMETA-ABDELLA
Felmeta Abdella (full name Felmeta Abdella Abrehim; Oromo: Falmataa Abdallaa; ግእዝ፡ ፈልመታ አብደላ; born December 12, 2004) is an Ethiopian political activist and a member of the Oromo Federalist Congress (OFC) political party.Birth: Felmeta Abdella was born on December 12, 2004, in Ijaginna, Hararghe Province, Ethiopia, an area bordering Harari
